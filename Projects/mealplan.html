Meal Planner Web Application

  • Project Summary:
    • Developed a web application that simulates a meal planner, allowing users to find recipes, create weekly meal plans, and interact with a community of food enthusiasts
  • Technologies Used:
    • HTML, CSS, JavaScript, PHP, MySQL, Tasty API integration
  • What I accomplished:
    • Developed a role-based system, allowing Admins to manage system data through the web UI
    • Designed an intuitive recipe search and pinning system
    • Enabled users to create custom recipes with ingredients, steps, and images
    • Implemented user profiles with public and private views
    • Developed a weekly meal planner with extra recipe suggestions
    • Fetched and cached recipe data from an external API
    • Implemented a user review and rating system for recipes
    • Featured trending recipes and verified users' meal plans on the main page
